# 钻石科普文案 - 2026年3月23日

---

## 📌 第一篇：钻石是如何形成的？

💎你知道吗？你手上那颗闪闪发光的钻石，诞生于几十亿年前！

🌍**地下深处**：钻石形成于地表以下150-200公里的地幔层，温度高达1050-1200°C，压力相当于5万大气压。

⏳**超长时间**：这个过程需要10亿-33亿年！比恐龙出现还要早几十亿年。

🌋**来到地表**：火山喷发时，岩浆会将钻石带上来，这就是为什么大多数钻石矿都在古老的火山管道附近。

💎所以每一颗钻石，都是地球漫长岁月的结晶✨

## 📌 第二篇：4C标准，看懂钻石价值！

买钻石不再迷茫，记住这4个C👇

💠**Carat 克拉**：重量，1克拉=0.2克，越大越稀有

💠**Cut 切工**：工艺，切工越好越闪，切工是钻石的灵魂！

💠**Color 颜色**：D色最白，Z色最黄，D-H色最受欢迎

💠**Clarity 净度**：内含物多少，FL/IF最干净，SI以下肉眼可见瑕疵

💡**划重点**：切工是最重要的C！一颗切工差的钻石，再大也不闪！

## 📌 第三篇：钻石产地知多少？

🌍全球钻石主要产地分布👇

🥇**俄罗斯**：全球最大钻石生产国，占产量约35%

🥈**博茨瓦纳**：非洲最大钻石生产国，品质很高

🥉**加拿大**：近年崛起，以高品质著称

🇦🇺**澳大利亚**：以粉钻和黄钻闻名

💎**印度**：历史最悠久的钻石产地，最早的钻石矿之一

💡**小知识**：不是产地越有名钻石越好，切工和4C才是关键！产地只是加分项～
