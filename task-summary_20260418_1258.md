# 任务：扩展旅游攻略为三级导航

## 目标
将原有8城市旅游攻略扩展为：中国 → 省份 → 城市 → 景区路线

## 实现方案

### 网站结构
- **第一级**：中国（选择省份）
- **第二级**：省份页面 → 城市列表
- **第三级**：城市详情 → 每日路线 + 景点推荐

### 覆盖范围
| 省份 | 城市 |
|------|------|
| 四川省 | 成都、九寨沟、乐山 |
| 云南省 | 丽江、大理、昆明、西双版纳 |
| 浙江省 | 杭州、乌镇 |
| 陕西省 | 西安、华山 |
| 广西 | 桂林、阳朔 |
| 福建省 | 厦门、泉州、土楼 |
| 黑龙江 | 哈尔滨、雪乡 |
| 重庆 | 重庆市区、武隆 |

**总计**：8省份，18城市

### 每个城市数据
- 城市名称、标语、封面图
- 建议天数、人均预算、最佳温度、推荐季节
- 每日行程路线（时间+景点+描述+标签）
- 必去景点列表

## 技术实现
- 单页应用（SPA）架构
- 三级导航：顶部省份标签 + 城市标签 + 面包屑
- 响应式设计（支持手机端）
- 杂志风格设计保持不变
